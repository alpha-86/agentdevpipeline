# 006 - 2026-04-09 项目组合与恢复摘要执行层差距审计

## 结论

当前 AgentDevPipeline 已经有：

- 项目组合模板
- issue routing workflow
- context recovery workflow
- context recovery template

但还缺“运行态示例资产”，也缺 workflow 到模板和目录的绑定说明。

## 主要差距

### 1. 项目组合还停在表格模板

缺少：

- 实际可参考的组合视图示例
- 审计结果和项目组合视图的联动示例

### 2. 恢复摘要缺少实际样例

当前虽然有恢复模板，但没有实际示例，用户不容易理解“恢复摘要”应该写到什么粒度。

### 3. 审计报告缺少实际样例

本轮前面已经补了模板，但还缺可直接对照的记录样例。

## 本轮决策

1. 为项目组合视图补示例文件
2. 为上下文恢复补示例文件
3. 为审计报告补示例文件
4. 在 issue routing 和 context recovery workflow 中补模板与输出目录
