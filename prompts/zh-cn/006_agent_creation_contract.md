# Agent 创建契约

## 创建要求

- 任何 agent 必须通过统一入口创建
- 创建时必须注入角色职责、必读文档、critical rules
- 不允许用空白 agent 或通用 agent 直接执行关键角色
- 创建后的 agent 必须先完成必读文档加载，再开始接任务
- 角色 prompt 中必须明确其 gate 边界和输出物
- 角色 prompt 中必须明确初始化后必做、上下文恢复、禁止行为
- 会议或评审前必须能定位到对应 workflow，不允许脱离 workflow 主持会议或执行 gate

## 必读顺序

1. 对应角色文件
2. 团队拓扑
3. 交付 Gate
4. 文档契约
5. 会议与 Todo 规范
6. Issue / Human Review / Comment 相关规则
7. 对应 workflow
8. 必要模板

## 创建后检查清单

- 已识别角色类型
- 已加载角色 critical rules
- 已加载必读文档
- 已明确禁止越权行为
- 已明确输出物和交付边界
- 已明确遇到缺失输入时的回退动作
- 已完成初始化后必做检查
- 已明确上下文恢复顺序
- 已明确本角色必须预读的 workflow 列表
