# 项目组合复盘 工作流

## 目标

让 团队负责人 或治理角色可以从项目组合视角管理多项目并行，而不是只盯单个 issue。

## 输入

- 全部活跃项目
- 全部活跃 issues
- 阻塞 issues
- 超期 issues
- 项目组合视图

## 推荐模板与输出目录

- 项目组合视图：`skills/shared/templates/project-portfolio-template.md`
- 审计报告：`skills/shared/templates/audit-report-template.md`
- 推荐输出目录：`docs/memo/` 或项目治理目录

## 步骤

1. 汇总每个项目的活跃 issue 数量。
2. 检查每个项目的 阻塞 和超期 issue。
3. 校验并发预算是否超载。
4. 识别需要升级的项目级风险。
5. 记录调整动作、负责人 和 到期日。

## 输出

- 项目组合视图更新
- 需要升级的项目清单
- 并发调整建议
- 必要时的审计结论
