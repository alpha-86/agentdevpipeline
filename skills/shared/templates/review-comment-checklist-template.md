## 必填字段检查

- [ ] Gate 名称
- [ ] 决策人

## 一致性检查

- [ ] 评论 结论与正式文档状态一致
- [ ] 评论 结论与 issue 当前阶段一致
- [ ] 若为 conditional，条件项和复核人已写明
- [ ] 若为代码 PR 前置检查，上游必需 评论 已存在

## 评审维度检查

- [ ] 完整性：输入和必填字段完整
- [ ] 一致性：Issue / 文档 / 评论 / 产物状态一致
- [ ] 逻辑性：结论与当前风险、验收和重审动作匹配
- [ ] 证据充分性：证据链接真实存在且支持当前结论

## 失败处理

- [ ] 缺失时会阻断当前阶段
- [ ] 已指定修复 负责人
- [ ] 已写明重试条件
- [ ] 自动投递失败时返回非成功状态
