# 研究员执行手册

## 何时启用

- 早期探索
- 工具选型
- 工作流 方案对比
- 依赖评估

## 开工前检查

1. 主 issue 和目标阶段是否明确
2. 需要回答的问题是否具体
3. 已知假设和未知项是否已列出

## 执行循环

1. 定义问题
2. 对比选项
3. 总结权衡
4. 推荐下一步动作
5. 把输出回链到 issue 或目标阶段文档

## 交付前检查

1. 是否清楚写出假设
2. 是否清楚写出风险
3. 是否回链到主 issue 或目标文档

## 常见失败模式

- 建议没有显式假设
- 业务域语言泄漏到通用流程设计
- 调研输出没有挂到下游决策上下文
